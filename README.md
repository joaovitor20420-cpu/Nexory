# Nexory - Digital Cockpits & Software Development 🚀

Uma landing page premium, moderna e de alta performance desenvolvida para a agência **Nexory**, especializada em arquitetura de softwares sob medida, foco em lucros, escabilidade e eliminação de *legacy debt*.

Desenvolvida com foco extremo em UX/UI, esta página utiliza conceitos modernos de web design como **Glassmorphism**, animações avançadas (*glow effects*, *magnetic buttons*, *scroll reveals*), além de ser totalmente otimizada para SEO e acessibilidade.

## 🎯 Destaques do Projeto

- **Design Premium:** Fundo escuro abstrato com texturas radiais e luzes neons (Ciano, Rosa e Roxo).
- **Glassmorphism:** Componentes translúcidos com efeitos de desfoque (*backdrop-filter*) e bordas dinâmicas.
- **Animações Magnéticas:** Botões e cards interativos que acompanham levemente o movimento do mouse para sensação de profundidade 3D.
- **Micro-interações:** Efeitos de *shimmer* (brilho rápido), bordas com gradiente, levitação (*hover lifts*) e tipografia cinética.
- **Scrollers Infinitos:** Carrossel horizontal suave, totalmente responsivo e com suporte a parada em *hover* na seção de Tecnologias (Tech Stack) e Processo (Como Funciona).
- **Sem Frameworks Pesados:** O front-end da landing page foi construído utilizando Vanilla CSS/JS puro para máxima velocidade de carregamento e nota máxima em *Core Web Vitals*.
- **Otimizada para Mobile:** Layout e arquitetura flex estruturados para celulares, tablets e desktops de alta resolução.
- **Call-to-Action Eficiente:** Redirecionamento direto com mensagens pré-configuradas para o WhatsApp.

## 🛠️ Tecnologias Dominadas / Stack da Agência

O escopo da **Nexory** abrange as mais modernas e sólidas tecnologias do mercado. Embora esta Landing Page seja ultra leve (escrita em Vanilla JS e CSS), a agência domina o seguinte stack:

*   **[HTML5](https://developer.mozilla.org/pt-BR/docs/Web/HTML)** - Estruturação semântica e SEO amigável.
*   **[CSS3](https://developer.mozilla.org/pt-BR/docs/Web/CSS)** - Estilizações nativas, animações de alta fidelidade e layouts complexos.
*   **[JavaScript](https://developer.mozilla.org/pt-BR/docs/Web/JavaScript)** - Dinamismo, lógica assíncrona e manipulação do DOM.
*   **[TypeScript](https://www.typescriptlang.org/)** - Superset para código tipado, seguro e altamente manutenível.
*   **[React](https://react.dev/)** - Criação de interfaces ricas, SPA (Single Page Applications) e componentização.
*   **[Next.js](https://nextjs.org/)** - O framework principal para aplicações React focado em SSR, SSG e máxima performance.
*   **[Node.js](https://nodejs.org/)** - Backend ágil e escalável utilizando JavaScript no servidor.
*   **[Tailwind CSS](https://tailwindcss.com/)** - Framework CSS de *utility-first* para acelerar o desenvolvimento front-end com padronização.
*   **[Git](https://git-scm.com/)** - Versionamento de código rigoroso e seguro.
*   **[GitHub](https://github.com/)** - Ecossistema de repositório, CI/CD e revisão de código.

## ⚙️ Como executar localmente

Como o projeto desta Landing Page não utiliza bundlers (como Vite ou Webpack) de forma intencional para manter sua simplicidade crua, executá-lo é extremamente fácil:

1. Faça o clone ou o download do projeto.
2. Navegue até o diretório `Nexory/`.
3. Abra o arquivo `index.html` em qualquer navegador web moderno (Google Chrome, Firefox, Edge, Safari).
    - *Opcional:* Você pode utilizar a extensão **Live Server** (do VS Code) para ter *hot-reload* contínuo durante as edições visuais.

## 📂 Arquitetura do Projeto

*   `index.html`: A estrutura semântica da página, abrigando *meta tags*, fontes estáticas via Google Fonts, CDN Icons (Devicon) e o sistema principal.
*   `style.css`: A "magia visual". Possui variáveis CSS no topo (Design Tokens), grid system flexível, e está modularizado internamente com comentários para cada seção (Navbar, Hero, Grid Bento, Infinite Scrollers, Offer, etc).
*   `script.js`: Arquivo JS extremamente enxuto que controla interceptações de scroll (observer), movimentações magnéticas complexas (mouse-tracking) e iluminação de cards sensível ao mouse (*glow effect*).
*   `images/`: Pasta reservada para ativos estáticos otimizados (logos, SVGs, cartões de demonstração e og-images).
